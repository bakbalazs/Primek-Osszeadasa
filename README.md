# Prímszámok összeadása JAVAFX-ben
![Alt text](https://github.com/bakbalazs/Primek-Osszeadasa/blob/master/kepek/fokepernyo.png "A program fõképernyõje")

### Funkció
* 1-5000 -ig megadhatunk számot és összeadja a megadott számú prímszámokat

### Példák
* Ha a beadott szám 3 akkor az eredmény 10 (2+3+5)
* Ha a beadott számm 100 akkor az eredmény 24133
* Ha a beadott szám 1000 akkor az eredmény 3682913

### Telepítés

##### Elsõ módszer
1. A tároló leklónozása
2. A leklónozott tároló _PrimekOsszeadasa_ mappájában adjuk ki a következõ parancsot: _mvn package_
3. Miután lefutott a parancs, a _Primek-Osszeadasa_ mappában adjuk ki a _java -jar PrimekOsszeadasa.jar_ parancsot
4. Az alkalmazás indul.

#### Második módszer
1. A _PrimekOsszeadasa.jar_ letöltése a tárolóból.
2. Dupla kattintás a _PrimekOsszeadasa.jar_ állományon az alkalmazás elindítását eredményezi.
